## 🔑 核心决策速查

| 维度 | 决策 |
|------|------|
| 产品定位 | 无障碍工具 + 创意教育，面向肢体障碍人士和儿童 |
| 绘图模式 | 混合：简单几何 → Canvas，复杂场景 → Qwen-Image-2.0 |
| ASR | Qwen3-ASR-Flash via DashScope |
| 意图理解 | Qwen3.7-Max + Zod Schema 结构化输出 |
| 触发逻辑 | AI 解析意图后弹框，用户语音选"1"或"2"确认 |
| 账户系统 | Supabase Auth + Prisma User 同步 |
| UI 风格 | 白底黑字 + 马卡龙蓝/樱花粉装饰，WCAG 2.1 AA |
| 部署 | Vercel（前端+API）+ Supabase（数据库+存储+认证）|
