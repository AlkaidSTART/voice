## 1. 产品概述

### 1.1 产品名称

**VoiceCanvas**（语音画板）

### 1.2 一句话定位

> 一款面向肢体障碍人士与儿童的纯语音驱动绘图工具——不需要鼠标，不需要键盘，只需开口说话，就能创作属于自己的画作。

### 1.3 产品背景

在全球范围内，约有 2.85 亿视力障碍者之外，还有数以亿计的肢体运动障碍人群（脑瘫、渐冻症、高位截瘫等）无法正常使用鼠标和键盘。传统绘图软件的交互模式对他们完全封闭。与此同时，儿童美术教育场景中，语音引导创作可以极大降低操作门槛，激发想象力。

本项目通过「语音 → AI 意图理解 → Canvas 绘制 / AI 生图」的混合架构，让创作权还给每一个人。

## 2. 目标用户

### 2.1 主要用户群体

| 群体 | 特征 | 核心痛点 |
|------|------|---------|
| **肢体障碍成年人** | 手部运动障碍，无法使用鼠标/触控 | 传统绘图工具完全不可用 |
| **3~10 岁儿童** | 识字量有限，精细动作不稳定 | 复杂 UI 难以操作 |
| **特殊教育学生** | 语言/认知发育不均衡 | 需要高容错、强反馈的交互 |
| **创意探索者** | 想快速把脑中想法变成图像 | 手绘效率低，AI 生图提示词复杂 |

### 2.2 用户画像

**画像 A · 小芽（7 岁，幼小衔接学生）**  
喜欢看绘本，想画出自己脑海里的故事，但对电脑很陌生。对她来说，说一句"画一只粉色的兔子"就能看到图案出现，会让她觉得这是魔法。

**画像 B · 林哥哥（28 岁，渐冻症患者）**  
曾是设计师，患病后手部功能逐渐退化。他依然有创作欲望，迫切需要一个能理解他语音指令的专业绘图工具，而不是只能说"好的"的语音助手。

## 4. 核心功能规划

### 4.1 功能全景

```
VoiceCanvas
├── 🎤 语音输入层
│   ├── 实时语音录制（Mic Button）
│   ├── ASR 转录（Qwen3-ASR-Flash）
│   └── 字幕实时回显
│
├── 🧠 意图理解层
│   ├── 语义分析（Qwen3.7-Max）
│   ├── 指令分类（Canvas / AI生图 / 控制指令）
│   └── 参数提取（颜色/位置/尺寸/形状）
│
├── 🎨 双模绘图引擎
│   ├── Canvas 模式（几何图形/基础绘图）
│   └── AI 生图模式（Qwen-Image-2.0）
│
├── 🖼 画布管理
│   ├── 图层叠加
│   ├── 撤销/重做栈
│   └── 导出（PNG/JPEG）
│
└── 👤 用户系统
    ├── Supabase Auth 登录
    ├── 作品历史保存
    └── 语音操作日志
```

## 5. 用户故事

### Epic 1：基础绘图（Canvas 模式）

- **US-01**：作为用户，我说"画一个蓝色的圆形"，画布上应出现一个蓝色圆形，并有字幕回显我说的内容。
- **US-02**：作为用户，我说"撤销"，上一步操作应被还原。
- **US-03**：作为用户，我说"清空画布"，确认弹框后画布清空。
- **US-04**：作为用户，我说"在中间画一个大红色矩形"，系统应在画布中央绘制大尺寸红色矩形。

### Epic 2：AI 场景生图

- **US-05**：作为用户，说完一段描述性语句（如"一片阳光下的樱花林"），系统询问"是绘制图形还是 AI 生成？"，我选择 AI 生成后，图片出现在画布上。
- **US-06**：作为用户，生成的图片可以进一步用 Canvas 指令在上面叠加图形（图层叠加）。

### Epic 3：作品保存与历史

- **US-07**：作为登录用户，我说"保存"，作品自动保存到我的图库，下次可查看。
- **US-08**：作为用户，说"导出"，画布以 PNG 下载到本地。

### Epic 4：无障碍交互

- **US-09**：作为视力正常但手部障碍的用户，所有操作均可仅凭语音完成，无需任何鼠标/键盘。
- **US-10**：作为儿童用户，界面有语音提示引导，麦克风按钮醒目，颜色温和不刺眼。

## 6. 非功能性需求

| 指标 | 目标值 | 说明 |
|------|--------|------|
| ASR 首字延迟 | < 1.5s | 语音结束到转录出现 |
| 意图识别耗时 | < 2s | LLM 响应时间 |
| Canvas 绘制延迟 | < 200ms | 本地操作，近实时 |
| AI 生图延迟 | < 15s | 可接受范围，展示进度条 |
| 语音识别准确率 | > 90% | 普通话标准口音 |
| 无障碍评分 | WCAG 2.1 AA | 颜色对比、焦点管理 |
| 移动端兼容 | iOS Safari / Android Chrome | 响应式布局 |

## 8. 版本规划

### MVP（黑客松 3 天）
- 核心语音 → Canvas 绘图流程
- AI 场景生图（Qwen-Image-2.0）
- 意图弹框确认
- Supabase 登录 + 作品保存
- 响应式 Web 端

### v1.1（黑客松后 2 周）
- 元素选中与移动语义理解
- 多图层支持
- 语音引导教程（新手）
- 离线 PWA 基础支持

### v2.0（长期）
- 多语言支持（粤语、英语）
- 手写板叠加（语音 + 辅助手写）
- 教育版（教师作品模板）
- 实时协同（多用户共同创作）
